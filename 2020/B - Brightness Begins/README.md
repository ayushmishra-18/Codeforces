## Problem Statement

<div class="header"><div class="title">B. Brightness Begins</div><div class="time-limit"><div class="property-title">time limit per test</div>1 second</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Imagine you have $$$n$$$ light bulbs numbered $$$1, 2, \ldots, n$$$. <span class="tex-font-style-bf">Initially, all bulbs are on</span>. To <span class="tex-font-style-it">flip</span> the state of a bulb means to turn it off if it used to be on, and to turn it on otherwise.</p><p>Next, you do the following:</p><ul> <li> for each $$$i = 1, 2, \ldots, n$$$, flip the state of all bulbs $$$j$$$ such that $$$j$$$ is divisible by $$$i^\dagger$$$. </li></ul><p>After performing all operations, there will be several bulbs that are still on. Your goal is to make this number exactly $$$k$$$.</p><p>Find the smallest suitable $$$n$$$ such that after performing the operations there will be exactly $$$k$$$ bulbs on. We can show that an answer always exists.</p><p>$$$^\dagger$$$ An integer $$$x$$$ is divisible by $$$y$$$ if there exists an integer $$$z$$$ such that $$$x = y\cdot z$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows.</p><p>The only line of each test case contains a single integer $$$k$$$ ($$$1 \le k \le 10^{18}$$$).</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output $$$n$$$ — the minimum number of bulbs.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div class="title">Input<div title="Copy" data-clipboard-target="#id006444056654596282" id="id00397213653114572" class="input-output-copier">Copy</div></div><pre id="id006444056654596282"><div class="test-example-line test-example-line-even test-example-line-0">3</div><div class="test-example-line test-example-line-odd test-example-line-1">1</div><div class="test-example-line test-example-line-even test-example-line-2">3</div><div class="test-example-line test-example-line-odd test-example-line-3">8</div></pre></div><div class="output"><div class="title">Output<div title="Copy" data-clipboard-target="#id00273298851539889" id="id002481012463460175" class="input-output-copier">Copy</div></div><pre id="id00273298851539889">2

</pre></div></div></div><div class="note"><div class="section-title">Note</div><p>In the first test case, the minimum number of bulbs is $$$2$$$. Let's denote the state of all bulbs with an array, where $$$1$$$ corresponds to a turned on bulb, and $$$0$$$ corresponds to a turned off bulb. Initially, the array is $$$[1, 1]$$$.</p><ul> <li> After performing the operation with $$$i = 1$$$, the array becomes $$$[\underline{0}, \underline{0}]$$$. </li><li> After performing the operation with $$$i = 2$$$, the array becomes $$$[0, \underline{1}]$$$. </li></ul><p>In the end, there are $$$k = 1$$$ bulbs on. Initially, the array is $$$[1, 1, 1, 1, 1]$$$. </p><ul> <li> After performing the operation with $$$i = 1$$$, the array becomes $$$[\underline{0}, \underline{0}, \underline{0}, \underline{0}, \underline{0}]$$$. </li><li> After performing the operation with $$$i = 2$$$, the array becomes $$$[0, \underline{1}, 0, \underline{1}, 0]$$$. </li><li> After performing the operation with $$$i = 3$$$, the array becomes $$$[0, 1, \underline{1}, 1, 0]$$$. </li><li> After performing the operation with $$$i = 4$$$, the array becomes $$$[0, 1, 1, \underline{0}, 0]$$$. </li><li> After performing the operation with $$$i = 5$$$, the array becomes $$$[0, 1, 1, 0, \underline{1}]$$$. </li></ul><p>In the end, there are $$$k = 3$$$ bulbs on. We can also show that the answer cannot be smaller than $$$5$$$.</p></div>
